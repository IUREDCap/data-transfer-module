This directory contains resources used by the Data Transfer external module, and includes:

    * CSS files
    * images
        * PowerPoint files used to create the images
    * JavaScript files


To convert a PowerPoint slide to an image, in PowerPoint (generally on Windows):

1. Go to the specific slide that has the image you want
2. Select **File -> Save As**
3. Enter the file name (without extension) shown in the slide notes
4. Select the file format shown in the slide notes, for example, __JPEG File Interchange Format (*.jpg)__
5. Click on the **Save** button
6. For the "Which slides do you want to export?" dialog, press __Just This One__
7. Resize the image as indicated in the slide notes (if indicated)
8. In general, you will want to crop the image to remove the huge margins of the image
9. Transfer the image file and the PowerPoint file (if changed) to the **resources/** directory on your development VM
